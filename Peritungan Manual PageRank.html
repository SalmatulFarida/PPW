Peritungan Manual PageRank#

PageRank disebut sebagai algoritma pencarian yang digunakan oleh Google Search untuk memberi peringkat halaman web, dalam mengukur nilai pada setiap halaman web dari berbagai faktor. Sebuah halaman bisa dikatakan sebagai halaman penting ketika halaman tersebut banyak terhubung dengan halaman lainnya.

Ilustrasi Sederhananya :

Ilustrasi diatas menunjukkan suatu bobot penting suatu halaman dengan adanya nilai yang muncul dan panah menggambarkan hyperlink atau banyak sedikitnya keterkaitan antar halaman.

Rumus PageRank :

Keterangan :

( PR(u) ) adalah PageRank dari halaman ( u ).

( d ) adalah damping factor, biasanya diatur sebagai 0.85.

( N ) adalah jumlah total halaman dalam graf (jumlah node dalam graf).

( B_u ) adalah himpunan halaman yang memiliki tautan ke halaman ( u ).

( L(v) ) adalah jumlah tautan keluar dari halaman ( v ).”
